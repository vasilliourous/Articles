                        Home Assistant
               

            Turning my PC into the ultimate smart home hub
    
Home Assistant is one of the first tools I set up on every new PC because it quietly connects everything I use into one system. I primarily use it for automating my environment, including lights, devices, and small daily actions that remove friction from my workflow.
PC monitoring
 is also one part of that setup. I track things like CPU usage, RAM, temperatures, and whether my system is idle or active. But instead of just looking at dashboards, I use this data to trigger automations. If my PC goes idle, certain things will turn off. When I’m back, everything resumes.
That’s what makes it powerful for me. It’s not just about smart home control or system stats, it’s about creating a setup that reacts to how I actually use my PC. Once it’s configured, it runs in the background and just makes everything feel smoother.

                        ClipCascade
               

            Private way to sync your clipboard history
    
Credit: 
One of the most annoying parts of moving between different PCs or laptops is losing my clipboard history. That’s where Clipboard Manager helps. Clipboard manager is one of the 
most underrated productivity tools
 for any productive setup. ClipCascade is one such self-hosted clipboard manager that keeps all my copied text, links, and snippets synced across every machine I use. Instead of emailing myself a block of code or a long URL, I just copy it on one device, and it’s instantly available on the other.
What I like the most is the focus on privacy and control. Unlike cloud-based clipboard sync tools, my data stays on my own hardware. The interface is clean and stays out of the way until I need to pull up something I copied an hour ago. For my workflow, it’s a massive time-saver, especially when I’m bouncing between writing a draft and testing a configuration. It’s one of those "set it and forget it" tools that I now can’t imagine working without.

    						ClipCascade
			
ClipCascade is a self-hosted clipboard manager providing secure, real-time sync across your devices. It ensures your text snippets and links stay private and accessible without relying on third-party cloud services.

                        Uptime Kuma
               

            Silent guardian for every service on my network
    
✕
 Remove Ads
Uptime Kuma is something I install early because I like knowing that everything I run is actually working. When you start self-hosting multiple services, things can silently break, and you don’t always notice until you need them.
I use 
Uptime Kuma to monitor all my services
 like local apps, Docker containers, and even a few external endpoints. It gives me a simple dashboard where I can instantly see what’s up and what’s down. If something goes offline, I get notified immediately so I can fix it before it becomes a problem.
The platform is simple and clean without any complicated setup. It just does one job really well. It’s one of those tools you don’t think about much, but once it’s running, you rely on it every day without even realizing it.

                        File Browser Quantum
               

            Modern way to manage your entire server filesystem
    
Managing files across different Docker containers and directories can quickly become a headache. That’s why File Browser Quantum is one of the first things I deploy on any new setup. It’s a lightweight, web-based file manager that essentially gives me a sleek, "Google Drive-like" interface for my server’s entire file system.
✕
 Remove Ads
What I love most is the sheer speed and the modern UI. It isn’t just about moving files around; it has excellent built-in preview support. I can glance at PDFs or stream media files directly in the browser without having to download them first. It also makes sharing incredibly easy; I can generate password-protected links with expiration dates in just a few clicks. Whether I’m quickly editing a config file for another app or organizing my latest research, having this centralized, responsive hub saves me from constantly jumping into a terminal or using clunky FTP clients.

    						FileBrowser Quantum
			
FileBrowser Quantum is a modern, self-hosted web file manager featuring indexed real-time search, granular access controls, multi-source support, and a responsive interface for seamless management.

                        Syncthing
               

            Smart background sync without relying on the public cloud
    
If there is one tool that saves me from the "where did I save that file?" panic, it’s Syncthing. 
Unlike cloud storage services
 that store your data on someone else’s server, Syncthing is a continuous file synchronization program that connects my devices directly. I use it to keep my most important folders, like my research notes and active blog drafts, perfectly in sync between my desktop, laptop, and home server.
