Like many people new to Linux, I started out using a graphical file manager for all my file tasks, such as opening folders, copying, moving, renaming, deleting, and searching. But once I got comfortable with basic terminal commands and began trying out terminal utilities, everything changed.

                        Zoxide
               

            Jump instantly based on history; no typing paths
    
Unlike in a GUI file manager, where you navigate to any location by clicking through multiple folders, in the terminal, using the classic cd command requires typing the full path. But if you're not a fan of typing long paths, everything changes once you 
start using zoxide
. With this tool, instead of clicking through folders or typing long paths, you can jump straight to where you want to be using just a few keywords.
The first few times you visit a directory, zoxide remembers it. Later, typing 
z
 projects instantly takes you to the right place, even if that folder is buried deep in your home directory. The more you use it, the smarter and more accurate it becomes, until navigating between directories feels completely effortless.
Zoxide is available on most Linux distributions through their default package managers. To get even more out of zoxide, you can integrate it with tools like fzf, Raycast, and many others.

                        Yazi
               

            Keyboard-driven navigation with image previews
    
If you've ever thought about using a terminal file manager, the first thing that comes to mind is that it might be complex and require memorizing multiple commands. But 
Yazi
 proves that the terminal doesn't have to be dull or limited. You get a full-screen file manager where you can navigate folders with the keyboard, open files effortlessly, and even preview images and videos right inside the terminal.
You can also browse screenshots, inspect other media, or view PDFs without launching external apps. The only minor drawback is that it takes a couple of hours to memorize the default keybindings, but you can customize them to suit your preferences anyway. To install Yazi on your Linux distribution, check its GitHub page for detailed instructions.

                        Eza
               

            A list that makes sense
    
✕
 Remove Ads
When listing files, you want clarity, not clutter. Eza, a modern, smarter alternative to the classic ls command, delivers exactly that. When you run 
eza
 in a directory, you get color-coded file types, icons (yes, right in your terminal), and Git status indicators all at once. You can immediately see which files have been modified, which are directories, which are executables, and which are tracked by Git.
It even includes a built-in tree view that reveals your directory structure without needing a separate tool. Eza is available in the default repositories of many Linux distributions, and it can also be installed via Cargo or prebuilt binaries. For the latest installation options and detailed instructions, check the official 
GitHub page
.

                        Fzf
               

            Instant fuzzy search to find anything
    
Fzf changes the way you search for things. Instead of remembering exact filenames or digging through folders, you type a few keywords, and fzf narrows down the results in real time. You select what you want and move on. This is incredibly useful for opening files, switching directories, or searching through command history.
You can pipe almost anything into fzf, such as the output from find, git branches, command history, or even zoxide directories. It also integrates seamlessly with your shell, which means you can search your entire command history by pressing 
Ctrl + R
 and typing a few letters. Although fzf is available on almost every Linux distro through the default package manager, you can still check the project's 
GitHub page
 for your preferred installation method.

                        Bat
               

            Syntax highlighting makes reading config/code easy
    
Reading files shouldn't require opening a GUI editor every time. Bat, a modern alternative to the classic cat command, lets you view files directly in the terminal with beautiful syntax highlighting and line numbers. It even includes Git integration to show recent changes. For example, displaying modifications in the margin (just like in VS Code), with added lines marked by a plus sign, removed lines by a minus, and modified lines highlighted differently.
