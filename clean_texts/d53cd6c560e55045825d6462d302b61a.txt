Warp just made running multiple AI agents dead simple for everyone. I’m now using Claude Code and Codex in one smooth agentic workflow
I tested Warp a while back and walked away impressed. It is one of the few AI coding tools I’d recommend to anyone serious about agentic development.
So when I came across their new 
Universal Agent Support update
, I had to try it immediately.
Here’s the problem most developers run into when using AI agents: you have one agent refactoring your code, another writing tests, and a third handling documentation.
The natural instinct is to run them one after another. That approach works, but it’s slow. You’re essentially lining up your agents in a queue when they could all be working at the same time.
The Warp’s Universal Agent Support is designed to solve this problem.
It lets you run Claude Code, Codex, Gemini CLI, and other coding agents side by side — each in its own tab, each working on a different task. These agents run in isolated Git worktrees, so their changes never conflict.
I pulled up an old test project that would be ideal for me to test and share with you.
