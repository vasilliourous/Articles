                        GIMP
               

            An image-editing alternative to Photoshop
    
 I'm far from a creative professional when it comes to image editing — I rarely work with layers. Most of what I do involves editing a few things like contrast and saturation, cropping photos, and censoring information in screenshots.
I used to have access to Photoshop when I was a student, but when that access ended, I switched to GIMP. While it lacks some of the features Photoshop has, you can install 
GIMP plugins to customize your experience
.

                        VLC
               

            No need for a paid media player
    
✕
 Remove Ads
 I've never used a paid media player for Windows, but there was a time when the OS's built-in player was woefully lacking and didn't work with many of the file formats I used. That's when I started using VLC as my go-to video player.
VLC has basic features but also supports a variety of file formats. I enjoy the fact that I can boost the video playback volume to over 100%. You can also use VLC as a streaming media player — both for self-hosted files and for connecting to remote servers.

                        7-Zip
               

            Compress files faster and for free
    
Many people forget that WinRAR is technically paid software. However, 7-Zip not only supports faster compression with its 7z file format, but it's also completely free and open-source. Most people get by just fine using Windows' built-in unzipping tools, but if you want to compress a large amount of data more quickly, then 7-Zip is a great option.
I also recommend 
using 7-Zip instead of WinRAR
 because WinRAR repeatedly prompts users to upgrade from the free trial. However, its support for more file formats and faster compression algorithms make 7-Zip the best choice for general usage and archiving data.

                        ShareX
               

            A feature-rich screenshot tool
    
Most of us don't need more than Windows' Snipping Tool for taking screenshots. But if you take a large number of screenshots or want to edit the image from within your capture software, then ShareX provides plenty of features while also being free.
I still rely on Snipping Tool for most of my screenshots, but when I need to add a bit more annotation or pixelate content, I switch over to ShareX. After all, it literally costs me nothing.

                        OBS
               

            Great for screen recording and streaming
    
I first started using OBS to record games for YouTube as it didn't require a subscription and included an array of features that made recording seamless. Over time, though, I've moved towards using OBS to record interviews and meetings as a backup to my notes.
