                        Penpot
               

            The designer-developer bridge you didn't know existed
    
Penpot is a fully open-source design and prototyping platform built by Kaleidos under the Mozilla Public License 2.0. You can use it in your browser, or self-host it on your own server. It handles UI design, interactive prototyping, component libraries, and developer handoff—essentially the same core workflow you'd use Figma for.
Apart from the usual array of tools and features you'd expect from a prototyping tool competing with Figma, one of the most useful Penpot features I found was CSS Grid Layout. It works exactly how it sounds as well: when you build a layout using Penpot's visual grid builder, the tool generates actual CSS grid code that developers can pull from the Inspect panel and drop straight into their projects.
Then you have native design tokens, developed in collaboration with the wider design-tokens ecosystem. Penpot is one of the first design tools to natively support the W3C design token specification. Your tokens for colors, spacing, and typography are stored in a standardized JSON-like format that moves freely between design tools and developer workflows.
✕
 Remove Ads
Penpot's developer handoff is also great. With Penpot's inspect tab, you get near production-ready code instantly. No plugins or payments required. Developers can open design files and extract what they need. Flexbox, CSS grid, spacing, typography, it all translates directly. This is one of the best tools on the market if you want to quickly convert design prototypes to fully-functional user interfaces. 
Figma's Dev Mode is useful for developers
, but it costs extra money. Penpot's Inspect tab is completely free.

                        Krita
               

            Seriously powerful digital painting
    
Krita has been one of the most popular free and open-source tools for digital artists for well over a decade now. It started as a basic paint program in 1999 and was essentially considered a GUI wrapper for ImageMagick for years. It runs on Windows, macOS, Linux, Android, and ChromeOS, and packs an OpenGL-accelerated canvas, non-destructive layers and masks, and an advanced brush engine that rivals software you'd be paying for. Even as a Photoshop rival, 
it is better than it has any right to be
.
✕
 Remove Ads
Krita's brush engine is its biggest strength. While Photoshop gives you basic brush customization, Krita offers nine different brush engines with over 100 professionally crafted presets. It's not just the sheer number of customization options and presets, either. The level of control can get overwhelming if you're migrating from Photoshop, and in a good way. Whether you're trying to simulate watercolors, oil paint, charcoal, or other brush types, Krita makes them feel natural.
There are also performance benefits you'll see between Photoshop and Krita that make the latter a much better tool, especially if you've got older hardware. Random hangs and crashes are far less common, and you're not slowed down by the program loading a tool or implementing some changes you made. You'll still need good hardware to use the program to its full potential, but Krita is more reliable and easier to run on weaker hardware compared to Photoshop.

                        Inkscape
               

            Vector work without the Adobe tax
    
Inkscape is an open-source vector editor that can handle everything Illustrator can do, including but not limited to drawing, shape tools, advanced path operations, text support, color management, and more, without the Adobe tax you pay monthly. At its core, Inkscape is built around SVG (Scalable Vector Graphics). While Illustrator uses its proprietary AI format and treats SVG as an export option, Inkscape uses SVG as its native format. The files you end up working on in Inkscape are literally the W3C standard code used by modern web browsers.
✕
 Remove Ads
Apart from your usual array of illustration tools, Inkscape also solves Illustrator's unnecessary need for switching tools for every action. For example, you need the Direct Selection Tool to adjust a point, but if you need to adjust a handle, the Pen Tool is required. Over the course of working on a design, that's a lot of tool switching. Inkscape handles this differently—you use the Node Tool instead.
You don't need multiple tools to select, move, or convert nodes. You just grab the path and drag it. If you want to change a curve, you can click the line segment itself and pull. It's far more tactile and fluid than Illustrator's segmented approach.

                        PhotoDemon
               

            The most portable Photoshop alternative you'll find
    
PhotoDemon is an open-source, completely free photo editing program for Windows that's been in active development since 2012. The entire application, with all of its features and functionality, weighs just 20.4 MB — hundreds of times smaller than Photoshop. It's the 
open-source Photoshop clone everyone keeps ignoring
 when they really should be paying attention.

PhotoDemon is what's called a portable application. It is Windows-only, but it doesn't require an installation. You can just download a ZIP archive from the official website, extract it, and you're ready to go. No registry entries, system dependencies, or administrator privileges required. It can live on a USB stick, sit on a portable drive, or even run from cloud storage if you want. Wherever you put it, it works the same way.
It's also not a glorified Paint replacement either. The program has over 200 editing tools built into it, including commonly used features like layers, masks, adjustment curves, content-aware fill, perspective correction, RAW file support, and even Photoshop PSD compatibility.

            Paid design tools aren't necessarily superior anymore
    
Dropping subscriptions doesn't mean you're downgrading your tools. Penpot, Krita, Inkscape, and PhotoDemon can easily cover most digital design and image work, regardless of whether you're prototyping an interface, painting a digital scene, working with vectors, or retouching photos.
