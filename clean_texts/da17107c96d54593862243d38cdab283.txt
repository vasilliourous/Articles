AerynOS 2026.05 Released with Linux Kernel 7.0, New Logo, and More
AerynOS 2026.05 Released with Linux Kernel 7.0, New Logo, and More
This release ships with GNOME 50.1 as the default desktop environment, as well as numerous updated components.
by 
Marcus Nestor
April 30th, 2026
April 30th, 2026
2 Comments
Share this article:
COPYRIGHT 9to5linux.com - DO NOT COPY WITHOUT PERMISSION
The AerynOS project released AerynOS 2026.05 today as the latest ISO snapshot of this independent Linux distro previously known as 
Serpent OS
.
The biggest change in the AerynOS 2026.05 release is that the underlying OS is now powered by the latest and greatest 
Linux 7.0
 kernel series. This should provide AerynOS with slightly faster performance, better hardware support, and various other enhancements.
One important thing to mention here is that, starting with this release, AerynOS now offers two alternative kernel options, namely linux-stable, which follows the latest stable kernel release with very few optimizations, and linux-gaming, which includes patches for handheld gaming and miscellaneous performance optimizations.
AerynOS continues to ship GNOME as the default desktop environment, and the new release uses the latest 
GNOME 50.1
 update. On the KDE Plasma front, the new AerynOS release ships with the 
KDE Plasma 6.6.4
 desktop environment featuring the latest 
KDE Frameworks 6.25
 and 
KDE Gear 26.04
 software suites.
Among other noteworthy changes, the 
moss search
 received support for using provider syntax, the ability to highlight matched substrings in the output by name or summary, and smarter grouping of results based on name or summary, 
moss state remove
 is now a lot faster, and there’s a new logo and new wallpapers.
Under the hood, AerynOS 2026.05 brings many updated components like COSMIC 1.0.11, Mesa 26.0.6, 
PipeWire 1.6.4
, 
QEMU 11.0
, Wine 11.7, Firefox 150.0.1, 
Thunderbird 150.0
, Python 3.14.4, Rust 1.95.0, Docker 29.4.1, Maven 3.9.15, ntpd-rs 1.7.2, zls 0.16.0, ly 1.3.2, Kmscon 9.3.5, eza 0.23.4, dash 0.5.13.3, fresh 0.2.23, LLVM 22.1.4, dankmaterialshell 1.4.6, and uutils-coreutils 0.8.0.
For more details, check out the 
release announcement page
. Meanwhile, you can download the AerynOS 2026.05 ISO from the 
official website
. However, please try to keep in mind that AerynOS is currently considered of alpha quality, so don’t install or use it in a production environment.

Not making any assumptions. Personally, I’ve upgraded three different machines (Intel and AMD) so far to Linux 7.0.x and they feel a bit faster overall. I also saw several reports from people on social media reporting better input latency (games feeling snappier) or increased battery life. Other news outlets reported faster PostgreSQL read/write performance and UDP network performance in tests. No regressions so far on my end with Btrfs, so maybe the distros I am using did a good job integrating Linux 7.0. Also, not using bootc.
